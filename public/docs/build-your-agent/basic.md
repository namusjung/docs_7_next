{% section id="basic-settings" title="Basic Settings" %}

The Basic Settings section allows you to define the core identity and description of your chatbot agent. This is the first step in configuring a new agent.

### Configuration Fields

| Field | Description | Example |
|-------|-------------|---------|
| **Agent Name** | The internal name of the chatbot agent. This is visible only to admins within the platform and is useful for organizing multiple agents. | `Customer Support Agent` |
| **Chatbot Display Name** | The public-facing name shown to end-users in the chat widget. This is how your users will recognize the chatbot. | `AI Assistant` |
| **Description** | A short description of the agent’s purpose and role. This helps admins quickly identify the agent’s use case. | A new AI agent designed to handle customer support queries and FAQs. |

{% callout type="tip" title="Best Practices for Basic Settings" collapsible=true %}

| Setting | Best Practice |
|---------|---------------|
| **Agent Name** | Choose a clear and descriptive name for internal tracking. |
| **Display Name** | Set a friendly and approachable name that matches your brand voice. |
| **Description** | Use the description to summarize the agent’s purpose (e.g., support, lead generation, sales). |

{% /callout %}
